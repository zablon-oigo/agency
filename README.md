# Agency Landing Page
## Tech Stack

**Client:** React,  TailwindCSS
## Run Locally

Clone the project

```bash
  git clone git@github.com:zablon-oigo/agency.git
```

Go to the project directory

```bash
  cd agency
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
